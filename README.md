# akihitoshinozaki.github.io

Personal website of Akihito Shinozaki — CS & Data Science @ University of Minnesota.

Static site (HTML/CSS/JS), hosted on GitHub Pages.

## Deploy

```bash
git clone https://github.com/akihitoshinozaki/akihitoshinozaki.github.io.git
# copy these files into the repo, then:
git add .
git commit -m "Launch personal site"
git push origin main
```

Live at https://akihitoshinozaki.github.io within a minute or two.

## Structure

- `index.html` — single-page content
- `style.css` — design tokens and layout
- `script.js` — scroll-reveal animation
- `assets/headshot.jpg` — portrait
